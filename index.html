ESTE

Les EST (Ecoles Supérieures de Technologies) sont un réseau d’établissement dans diverses villes marocaines, les EST sont spécialisées dans la formation technique, la durée d’études à l’EST dure 2 ans (4 semestres). On compte des EST dans plusieurs villes : Oujda, Agadir, Casablanca, Meknes, Salé…etc.
